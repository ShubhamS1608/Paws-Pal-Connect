This project is built using **Next.js 16, Tailwind CSS, and React Server Components**.

It simulates a pet hotel dashboard where guests (pets) can be filtered by species using URL state.

---

## Features Implemented

- React Server Components (RSC) for initial data fetch
- Suspense-style loading skeleton
- URL state synchronization using `?species=dog`
- Search bar for species filter (dog, cat, bird)
- Reusable components:
  - SearchBar
  - StatusButton (Check In / Check Out)
  - DarkToggle (Dark mode support)
- Responsive Tailwind layout (1 column mobile, 3 columns desktop)

---

##  Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS
- TypeScript
